
# Contributing to Awesome AR/VR/MR/XR

Thank you for your interest in contributing!
This project aims to maintain a high‑quality, well‑organized, and consistently formatted collection of AR/VR/MR/XR resources.
Please follow the guidelines below to ensure your contribution can be reviewed and merged smoothly.

## How to Contribute

1. Fork the Repository
2. Create a New Branch
3. Add or Update Content
4. Submit a Pull Request (PR)

## Contribution Rules

Allowed Contributions:
- Courses, tutorials, papers, datasets
- SDKs, frameworks, developer tools
- Books, videos, research groups
- Well‑written articles and industry reports
- Open‑source projects related to AR/VR/MR/XR
- Case studies and practical XR applications

Not Allowed:
- Paid ads or spam
- Low‑quality or outdated resources
- Unrelated content

## Formatting Guidelines
Use this structure:
- Resource Name — Short description (1–2 lines). URL

## Reporting Issues
Please open an issue with a clear title and description.

Thank you for contributing!
